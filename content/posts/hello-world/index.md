## 欢迎

这是我的第一篇博客文章！🎉

经过一番折腾，终于用 **Hugo** + **PaperMod** 主题搭建好了这个博客。

## 为什么要写博客？

1. **记录学习** — 把学到的知识整理成文字，加深理解
2. **分享交流** — 把自己的经验和思考分享给更多人
3. **个人品牌** — 持续输出有价值的内容，建立个人影响力

## 技术栈

- **框架**: Hugo（Go 编写的静态站点生成器）
- **主题**: PaperMod
- **部署**: GitHub Actions → 云服务器 Nginx
